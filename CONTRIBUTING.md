# Contributing to CodeAnalyzer

感谢你考虑为 CodeAnalyzer 做出贡献！

## Reporting Issues

如果你发现了 bug 或有功能建议，请创建一个 issue：

- **问题描述**: 清楚地描述问题
- **复现步骤**: 详细的步骤来重现问题
- **环境信息**: Python 版本、操作系统

## Submitting Pull Requests

1. Fork 项目
2. 创建分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m "Add feature"`
4. 推送分支：`git push origin feature/your-feature`
5. 创建 Pull Request

## Code Style

- 遵循 PEP 8
- 使用类型提示
- 编写 docstring

## Testing

```bash
python -m pytest tests/
```
